# 这个是使用tailwind.css技术的项目
# 运行说明
    npm install tailwindcss
    npm run build

# 项目简介
## 使用了tailwind.css+原生技术

    |──node_module （依赖文件下载）
    |──src  （资源文件）    
        |──assets      
            |──common.css   样式文件     
            |──img          图片    
        |──components       公共组件    
        |──pages            存放html页面文件     
        |──utils            公用js方法     
    |──package.json          
    |──tailwind.config.js      tailwind相关配置
